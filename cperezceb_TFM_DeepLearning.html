Deep Learning¶

Aplicacin de un modelo de Deep Learning para predecir la demanda de productos en funcin de una serie de factores contextuales y temporales. Compuesto por mltiples capas para aprender caractersticas complejas y de alto nivel directamente desde los datos sin necesidad de ingeniera manual de caractersticas.

Finalmente se evalua su eficiencia con el calculo de mtricas MSE, MAE y R.

Estructura del modelo

  1. Capa de Entrada: El modelo recibe mltiples caractersticas como entrada: variables numricas (media_X_dias y venta_lag_X_dias) y categricas (producto, abierto/cerraod, EsPromocin...)

  2. Capas Ocultas:

  • Capa dense_3 (con 128 neuronas): Es una capa totalmente conectada que transforma las caractersticas de entrada a un espacio de 128 dimensiones.
  • Capa dropout_2 (Dropout de 50%): Ayuda a prevenir el sobreajuste desactivando aleatoriamente el 50% de las neuronas durante el entrenamiento.
  • Capa dense_4 (con 64 neuronas): Otra capa densa que reduce la dimensionalidad antes de la capa de salida.
  • Capa dropout_3 (Dropout de 50%): Similar a la anterior, ayuda a prevenir el sobreajuste.

  1. Capa de Salida: La capa final que produce una sola prediccin, en este caso, el nmero de unidades vendidas (udsVenta).

  2. Optimizacin y Funcin de Prdida: El modelo utiliza el descenso de gradiente para minimizar la funcin de prdida. La funcin de activacin en las capas ocultas es ReLU, lo que permite que el modelo aprenda relaciones no lineales.

# Cargar el archivo CSV con los datos de ventas
file_path = "Data/VentasRdo.csv"

df = pd.read_csv(file_path, parse_dates=['idSecuencia'])

# Variables independientes y dependientes
X = df[['producto', 'bolOpen', 'EnPromocion', 'anyomes', 'diasemana', 'semana', 
        'media_7_dias', 'media_30_dias', 'venta_lag_1', 'venta_lag_7', 'venta_lag_30']]
y = df["udsVenta"]

# Transformar variables categricas y escalar
preprocessor = ColumnTransformer(transformers=[
    ("num", StandardScaler(), ['anyomes', 'media_7_dias', 'media_30_dias', 'venta_lag_1', 'venta_lag_7', 'venta_lag_30']),
    ("cat", OneHotEncoder(drop="first"), ['producto', 'bolOpen', 'EnPromocion', 'diasemana', 'semana'])
], remainder='passthrough')

X_processed = preprocessor.fit_transform(X)

# modelo
model = Sequential([
    Input(shape=(X_processed.shape[1],)),
    Dense(128, activation='relu', input_shape=(X_train.shape[1],)),
    Dropout(0.3),
    Dense(64, activation='relu'),
    Dropout(0.2),
    Dense(1, activation='linear')  # Activacin lineal para regresin
])

model.compile(optimizer=Adam(learning_rate=0.001), loss='mse', metrics=['mae'])
model.summary()
